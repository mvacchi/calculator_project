## Key Learnings

- How to manage UI state across multi-step interactions
- Implementing calculator logic without using `eval()`
- Handling edge cases like operator spam, chaining, and divide-by-zero
- Separating behavior (JavaScript) from layout and presentation (CSS)
- Mapping keyboard input to existing button logic without duplicating code
